#LoonTooth（笨牙)
![alt text][0]

基于**Android**的点对点文件传输，相当于集成了蓝牙的文件管理器，可以方便传输照片，音乐，视频，以及手机SD卡中的所有文件。
暂时只支持Android3.0以上设备。关于这个，我写过一篇博文：[《笨牙(LoonTooth)》][1]，你可以从这里了解更多。代码完全开源，如果有需要，可以邮箱和我联系。
